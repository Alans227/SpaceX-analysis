Исследование запусков ракетоносителей Falcon9 компании SpaceX в рамках курса 'Наука о данных IBM'. Все файлы данного репозитория являются частями выпускного задания этого курса.

В репозитории можно найти следующие jupyter notebooks:
1) Сбор информации с использованием Rest API и BeautifulSoup
2) Обработка данных(numpy, pandas, sqlalchimy)
3) Визуализация данных с использованием Matplotlib, Seaborn, Folium
4) Построение, подбор гиперпараметров и оценка точности для моделей классификации, таких как SVM, Logistic Regression, KNN и DecisionTreeClassifier.

Также в репозитории есть приложение на фреймворке Dash и презентация и результатами исследования.
